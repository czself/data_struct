### 2.指针（pointer）

#### 1）指针为什么要存在？

从本质上说，指针存在的目的是为了块状内存的传递。

```c
int a[10] = {0};
int b[10];
for(int i=0;i<10;i++){
	b[i] = a[i];
}

int *c = a;

// c 和 a指向的是同一块内存。
// 但是b不是。
```

#### 2）指针为什么很恶心？

```cpp
int a = 10;
a = 11;

int *p = &a;
*p = 12;
```

#### 3)别的语言怎么做？

下学期学习的Java表面上看没了指针，但是它依然存在。不过，Java隐藏的很好，它不让程序员直接改内存中的值。
