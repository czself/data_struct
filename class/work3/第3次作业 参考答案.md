## 3.找出列表中的素数。

我们用埃氏算法：

1. 建立2~N的数字列表
2. 从小到大，如果一个数字能被前面的数字整除，就把它删了。
3. 当执行到N的开方时，剩下的就是素数了。

如找出10以内的素数。

1. 建立数字列表[2,3,4,5,6,7,8,9,10]
2. 从2开始，凡是能被2整除（除2外）的就删了，剩下：[2,3,5,7,9]
3. 再删除3的倍数，剩下：[2,3,5,7]
4. 由于10的开方小于4，所以剩下的列表就是素数列表了。



```c
#define MAX 10000
int main(){
	int a[MAX] = {0};
	int i = 0,j = 2;
	for(;i < MAX;i++){
		a[i] = j++; 
	}
	/*注意，a里面存储的值是从 2~MAX+1 ，不过没关系，小问题*/

	
	for(i = 0;a[i]*a[i] <= MAX; i++){
		if(a[i] == 0){
			continue;
		} 
		for(j = i + 1;j < MAX;j++){
			if(a[j] % a[i] == 0){
				a[j] = 0;
			}
		}
	}
	for(i=0;i<MAX;i++){
		printf("%d,",a[i]);
	}
	
} 
```

所谓质因数，其实就是一个数字，它能分解成所有质数的乘积，如`100=2×2×5×5`——所有的因子都是质数。

```c
//这个题很简单
#define MAX 1000
int main(){

    int N = 0;
    int i = 2;

        if(N % i == 0){
```

## <font color="#d94f8a">5 完全数 </font>

除1外的一个正整数，如果所有因子之和（包含1，不包含本身）等于自身就叫完全数。

如数字6，它的因子有：1,2,3；1+2+3 = 6。所以6是完全数。

求10000以内的完全数。

```c
int prefect(int n){
    int i = 2;
    int sum = 0;
    for(;i <= n/2; i++){
        if(n % i == 0){
            sum += i;
        }
    }
    return sum +1; // 1是所有的数字的因子。
    
}
#define MAX 10000
int main(){
    int a[MAX] = {0};
    int i = 2 // 不包含自身，所以1 不是完全数
    int j = 0;
    for(;i<MAX;i++){
        if(prefect(i) == i){
            MAX[j++] = i;
        }
    }
    return 0;
}
```

## <font color="#d94f8a"> 7 黑洞数 </font>

大家知道宇宙中有一种叫“黑洞”的天体，是由高密度物质组成，连光线射到这个天体上都被吸收掉，不能反射，人们看不见这个天体，所以称它为黑洞。而 `123` 就是数字黑洞。我们取任一个数，如`81872115378`，其中偶数个数是` 4`，奇数个数是` 7`，是 `11` 位数，又组成一个新的数 `4711`。该数有 `1` 个偶数，`3`个奇数，是 `4` 位数，又组成新数`134`。再重复以上步骤，`1` 个偶数，`2 `个奇数，是 `3` 位数，便得到` 123` 黑洞。反复重复以上程序，始终是 `123`，就再也逃不出去，得不到新的数了。对任何一个数重复以上步骤都会得到 `123 `黑洞。
