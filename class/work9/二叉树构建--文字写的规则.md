1.构建一个struct：一定有int的data，还要有一个char op ,最关键的是连个指针——left right。 

2.读取string，依次读取，仿佛我们每个人都是一个i 

3.根据读取的内容，我们先构建一个struct node（是一个指针）。
	如果读取的字符是+ - *  / , 让struct中的op = 该字符。data 默认 -1
	如果读取的字符是数字，我们让struct中的op = '#'，用来表示，这个节点是一个数字，而不是一个操作符。同时，data = （转换成int类型）该字符 
	
--- 以上是根据字符串的每个字符，构建成不同的节点。下面要根据每个节点，把它正确地挂到树上去。

4.树有一个根节点，叫做 root，它是一个指针。

5.写一个函数 buildTree，它参数有两个，一个是root——它代表的其实一个二叉树。第二参数是我们刚才新建的节点（node）。返回值就是root指针。
	在buildTree函数中，判断node的op和data，主要是判断op
	
```cpp
①如果node->op == '#' ,表示它是一个数字，而不是操作符。按照规则，它要沿着root->right->right->...>NULL 的路线一直挂到最后的节点上。
	第一个节点不同，第一个节点挂载的时候，root==NULL，意味着，这个树还不存在，因此第一个节点理所当然地成root。
	
②如果 node->op == '+' 或者 '-',根据我们分析的规则，它一定是向上浮。原来的root成为它（新node）的 left，把自己变成新的root
		node->left = root;
		root = node;
		return root; 
③如果 node->op == '*' 或者 '/' ,根据我们分析的规则，先要判断root节点。
	（1）如图root节点的op == '#' 或者  '*' 或者 '/'，意味着这个新节点也要上浮。其规则与②相同。
	（2）如果root节点的op == '+' 或者'-',意味着，这个新节点要下沉。先将root的right指向自己的left，在将自己成为root的right。
	
		node->left = root->right;
		root->right = node;
		return root;
```
